To start the Application, 

Prerequisite: 
- You should have one Ide like Intellige, Eclipse to run this.
- Java
- Maven (The POM.XML file will fetch the Spring Boot web Jar from maven)

How to run this application?
- Go to the -> Inspection\src\main\java\com\example\inspection\InspectionApplication.java
- Run InspectionApplication class
- Open the browser (We have tested in chrome browser, it's working fine.)
- Run http://localhost:8080
